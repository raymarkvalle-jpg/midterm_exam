1. Ask for the player’s in-game name (IGN) and current rank (e.g., Epic, Legend, Mythic).
 
2. Display the hero roster using a loop.
 
3. Accept 4 match entries. For each match, ask the hero number (1–5). If the user types 0, that match slot is skipped (no stats recorded).
 
4. If the hero number is valid (1–5), ask for kills, deaths, assists, and match result (W for win, L for loss).
 
5. Compute KDA per match: KDA = (kills + assists) / deaths. If deaths is 0, use 1 as the denominator. Display KDA to 2 decimal places.
 
6. Assign a performance tag based on KDA and result: KDA ≥ 5 and W → 'DOMINATION!'; KDA ≥ 5 and L → 'Carried Hard'; KDA < 5 and W → 'Team Effort'; KDA < 5 and L → 'Better Luck Next Game'.
 
7. After all entries: count wins and losses among logged matches, compute win rate as a whole-number percentage (wins / matches_played × 100), and identify the match with the highest KDA as the Best Match. 
 
8. Print a formatted match log showing: IGN, rank, each logged match (sequential number, hero, KDA, result, tag), win/loss record, win rate, and best match label.
